## endereços

- [Demo](https://guicampanati.github.io/enderecos/)

### Sobre

Aplicação client-side onde é possível armazenar uma lista de endereços, sendo possível realizar operações CRUD. Nesta versão da aplicação, a lista é salva no localStorage do navegador. Se o usuário sair da página e voltar a lista é recuperada. Para cada endereço é possível visualizar a distância (em linha reta) da localização atual do usuário, clima no endereço e um link que abrirá um mapa com a rota para o endereço.
